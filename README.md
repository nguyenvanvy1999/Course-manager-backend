## Description
Courses manager built with [Nestjs](https://nestjs.com/), [Express](https://expressjs.com/) and [MongoDB](https://www.mongodb.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Technology stack

- Typescript

- Nest.js

- RESTful API

- Mongoose

- Socket.io
